# Hex color to UIColor

UIColor extension: convert Hex color to UIColor.

## Requirements

- Swift 4.0+

## Usage

#### Hex String to UIColor

Create colors using hex strings:

```swift
let color = UIColor(hex: "#7ED321")
// or
let color = UIColor(hex: "7ED321")
```
